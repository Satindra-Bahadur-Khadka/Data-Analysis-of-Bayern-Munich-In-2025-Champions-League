Champions League 2025: Bayern Munich & German Players Analysis
Project Overview

This project analyses the performance of Bayern Munich and German players during the 2025 UEFA Champions League season. It focuses on goals, attacking and defensive contributions, attempts, and goalkeeping statistics to provide insights into player and team performance.

Data Source

The datasets were sourced from Kaggle (uploaded by Pablo Ramos Wilkins) and include:

players_data.csv – Player demographics and club information

goals_data.csv – Goals scored, breakdown by type and location

attacking_data.csv – Assists, dribbles, corners, offsides

attempts_data.csv – Shots on/off target, blocked shots

defending_data.csv – Tackles, balls recovered, clearances

goalkeeping_data.csv – Saves, clean sheets, goals conceded, punches

teams_data.csv – Team names, country, and logos

Data Description

The datasets include player information (name, age, nationality, position, team), performance stats (goals, assists, attempts, defensive actions, goalkeeping), and team info (name, country, logo).

Analysis

Goals breakdown by player and type (foot, head, penalties)

Attacking stats: assists, dribbles, corners, offsides

Attempt stats: shots on/off target, blocked

Defensive stats: tackles, balls recovered, clearances

Goalkeeping stats: saves, clean sheets, goals conceded

Visualisations include bar charts and pie charts to highlight player and team performance

Tools Used

Python

Pandas for data manipulation

Matplotlib and Seaborn for visualisations

Jupyter Notebook for exploration and analysis


Objective

Provide insights into Bayern Munich and German players’ performance in the 2025 Champions League, highlighting key contributors and trends in attacking, defensive, and goalkeeping stats.
